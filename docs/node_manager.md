# BT Node Manager ユーザーガイド

`bt_node_manager.py` は、Behavior Tree のスキル開発を GUI で支援する統合ツールです。

## 主な機能

### 1. ノード作成 (Create)
*   **Action (動作)**: 時間のかかる処理（移動、掃除、発話など）に使用します。非同期通信です。
*   **Condition (判定)**: その瞬間の状態チェック（バッテリー残量、障害物の有無など）に使用します。同期通信（Service）です。
*   **自動コード生成**:
    *   C++ 側のファクトリ登録コード。
    *   Python 側のロジック雛形ファイル。
    *   Groot2 用のパレット XML 更新。

### 2. 管理 (Manage)
*   登録されているノードを一覧表示します。
*   不要になったノードを「完全に削除」できます。ソースコードや CMake の記述も自動でクリーンアップされます。

### 3. テスト (Test)
*   **これが最も強力な機能です。**
*   ツリーを組む前に、作成したノードを直接呼び出せます。
*   パラメータを入力して「Run Test」を押すと、目標（Goal）を送信したり、サービスを呼び出したりして、その応答（Feedback/Result）をリアルタイムで確認できます。

---

## 使い方フロー

1.  コンテナ内で `node_manager` を実行。
2.  「作成」タブで名前と引数を入力して生成。
3.  生成された Python ファイルに中身を書く。
4.  ビルド (`build`) して `run_logic` で起動。
5.  「テスト」タブで動作を確認。
6.  OK なら Groot2 でツリーに組み込む。
